Full-Stack Vanilla JS E-Commerce Engine
📌 Overview

The Full-Stack Vanilla JS E-Commerce Engine is a complete web-based shopping application built using pure JavaScript without any frameworks. It simulates real-world e-commerce functionality, including product browsing, cart management, and dynamic user interaction.

🚀 Features
🛒 Add to cart and remove from cart functionality
📦 Product listing with dynamic rendering
💰 Price calculation and cart total updates
🔄 Real-time UI updates using JavaScript
📱 Responsive and user-friendly design
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
📈 Functionality

The application allows users to browse products, add items to the cart, and manage their shopping experience seamlessly. All interactions are handled dynamically using JavaScript, ensuring a smooth and interactive experience without page reloads.

🎯 Purpose

This project was developed to demonstrate strong fundamentals in JavaScript, DOM manipulation, and full-stack logic implementation without relying on external libraries or frameworks.
